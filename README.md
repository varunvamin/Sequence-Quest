# Sequence Quest — Embedded C Project using a PSoC microcontroller

## Project Description
• Designed an embedded system using Embedded C on a PSoC microcontroller to validate and generate numerical sequences.
• Implemented keypad input, LCD interfacing, and UART communication for real-time sequence validation and display.

## Features
- **Mode 1: Sequence Quest**: Validates an initial sequence of 6 numbers, then automatically generates the next three numbers if valid.
- **Mode 2: Hex to BCD Conversion**: Converts hex digits entered from the 4x4 keypad to BCD and displays them.
- **Interfacing**: Hardware inputs map via HC-35S matrix keypad and UART output for simple terminal visualization.
